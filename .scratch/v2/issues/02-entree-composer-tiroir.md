# 02 — Entrée Composer : bouton + tiroir (patron Réglages), scène non grisée

**What to build:** un bouton **« Composer »** dans l'en-tête, à côté de
« Réglages », qui ouvre un **tiroir à droite** réutilisant exactement le patron
visuel du tiroir de réglages (mêmes hairlines, mêmes gabarits, glissée depuis la
droite). Différence clé : la **scène principale n'est pas grisée** — la portée
reste vivante et interactive derrière le tiroir. Le tiroir contient les outils de
construction (structure seulement dans ce ticket, câblage dans les suivants) :
sélecteur de **signature en pastilles**, les trois **Familles** (Notes / Silences
/ Modificateurs) en interrupteurs indépendants, la zone de palette, les actions
**Effacer / Tout effacer**, et le bouton primaire **« Jouer »**. Ouvrir Composer
démarre sur une **page blanche** (une mesure vide).
